# NeuraPal iOS App

Capacitor wrapper for the NeuraPal PWA — builds a native iOS app that loads the live web app.

## Project Structure
- `www/` — Web shell that redirects to the live PWA
- `ios/` — Native Xcode project (generated by Capacitor)
- `capacitor.config.ts` — Capacitor configuration
- `codemagic.yaml` — Cloud build configuration for Codemagic

## Build (via Codemagic — no Mac required)
1. Push this repo to GitHub
2. Connect to Codemagic.io
3. Add Apple Developer signing credentials
4. Run the `ios-release` workflow
5. IPA is built and submitted to TestFlight automatically
